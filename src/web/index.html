DocTranslator

Envie PDF/DOCX/PPTX/XLSX ou imagem (PNG/JPG) e baixe a tradução.
API /api/translate
Dica: para PDFs/DOCX/PPTX/XLSX o resultado é um arquivo traduzido. Para imagens, o resultado é um TXT. Se você enviar mais de 1 arquivo, o resultado será um translations.zip.
Prévia (quando for texto)
Imagens retornam texto; PDFs retornam arquivo.
